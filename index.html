cassandra by angosso

Mirror of Apache Cassandra (incubating); install schematool in debian package [rogerM]

Angosso Currently the only tie to Apache projects is the starter kit's use of the Ant build tool. There are potential future ties (http server, database backend, etc)that will be explored.

Dependencies

Running OS X have trouble getting Java 6 to work. setting to /System/Angosso.net/Frameworks/JavaVM.framework/Versions/1.6/Home.aspx and add /System/Library/Frameworks/JavaVM.framework/Versions/1.6/Home/bin to the beginning of /home/sites/angosso.net/home.aspx
